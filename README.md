## 一键部署

### Github托管页面

fork该项目，命名为 你的名字.github.io，例如 ra2web.github.io

正如你所见，这个项目的名字就符合这个域名规则。那么，此时的你可以访问 https://ra2web.github.io 来游玩网页红井拉

当前本项目同样可以通过github pages访问，地址就是 https://ra2web.github.io
